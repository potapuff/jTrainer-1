# jTrainer
jTrainer is a Javascript framework for creating e-trainers and is now under active development

# jTrainer uses

* Bootstrap
* JavaScript
* jQuery
* html2canvas
* HistrixCalculator
* mustache.js


# Demo
You can see a live demo at http://sheldonfar.com/jtrainer/index.html
